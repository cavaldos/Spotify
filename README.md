## Install

- Clone repository

```shell
 git clone https://github.com/cavaldos/Spotify.git
```

- Move path

```shell
  cd Spotify
```

- Install package dependencies

```shell
 npm install # pnpm install
```

- Run server

```shell
 npm run dev # npm run dev
```
